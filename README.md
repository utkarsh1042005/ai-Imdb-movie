# 🎬 AI Movie Insight Builder

An AI-powered web application that allows users to search for movies using an IMDb ID and receive **AI-generated audience sentiment insights** along with movie details.

This project combines **Next.js, Tailwind CSS, and Hugging Face AI models** to analyze audience reactions and present them in a clean, interactive interface.

---

## 🚀 Live Demo

🔗 **Live App:** https://ai-imdb-movie-79jx.vercel.app/

---

## ✨ Features

* 🔍 Search movies using **IMDb ID**
* 🎬 Fetch movie details (title, year, rating, cast, poster, plot)
* 🤖 AI-powered **audience sentiment analysis**
* 📊 Sentiment classification (Positive / Mixed / Negative)
* 🎨 Modern responsive UI
* ⚡ Fast API integration
* 🌐 Fully deployed on Vercel

---

## 🧠 How It Works

1. User enters an **IMDb movie ID**
2. The app fetches movie data from the **OMDb API**
3. Sample audience-style reviews are generated
4. Reviews are analyzed using a **Hugging Face AI model**
5. AI determines the **overall sentiment**
6. The UI displays:

   * Movie information
   * AI-generated audience insight
   * Sentiment result
